# WatchParty

Monorepo for the WatchParty platform.

## Workspace

- `apps/web`: Next.js frontend
- `apps/api`: NestJS backend API
- `packages/shared-types`: shared TypeScript contracts

## Quick Start

```bash
pnpm install
pnpm dev
```

## Common Commands

```bash
pnpm lint
pnpm build
pnpm check-types
```

Use Turbo filters when you need to scope work to a specific app or package.
